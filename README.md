# AI_Project

The Phase 1 report presents the first phase of the project focused on classifying museum environmental images into indoor and outdoor contexts using structural tree-based machine learning models. Utilizing a dataset consisting of 5,000 indoor and 5,000 outdoor museum images, we established an end-to-end pipeline involving three-channel spatial normalization, 150-component Principal Component Analysis (PCA) feature compression, and structural tree classification. We contrast a highly regularized Supervised Decision Tree with an eXtreme Gradient Boosting (XGBClassifier) and a transductive semi-supervised LabelSpreading framework. Our empirical results reveal distinct computational tradeoffs, demonstrating that while strict cost-complexity pruning stabilizes single-tree variance, ensemble gradient boosting optimizes interaction paths across high-dimensional visual tensors, and graph-based semi-supervised methods offer viable alternatives under restricted ground-truth data budgets.

The phase 2 report presents a deep learning solution for a binary image classification task using custom Convolutional Neural Network (CNN) architectures implemented in standard PyTorch. Facing challenges like spatial variance and overfitting, we employ offline and runtime data augmentations, including horizontal flips, random rotations, and color jittering. We evaluate a baseline custom CNN family by modulating depth (1-layer vs. 2-layer configurations) and learning rates (10⁻³ vs. 10⁻⁴). The dataset is dynamically processed with an automated train/validation pipeline using PyTorch's ImageFolder and DataLoader modules. Optimization is driven by the Adam optimizer minimizing Cross-Entropy Loss over 5 epochs per trial. Our multi-layer architecture paired with a tuned learning rate achieves the lowest verification loss and superior generalization. This demonstrates the critical impact of architectural capacity and hyperparameter tuning on spatial feature extraction.
